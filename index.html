CRUDYBeer

A full stack web app where users get to review beers. You are able to log in to view a large variety of beers, review them, and see reviews written by others. You are also able to edit or delete the reviews later.

Phaser Defense

A take on the classic archtype of Tower Defense. You have to fend off increasingly speedy pandas to defend the village. Each panda destroyed gets the user points and all the high scores are saved on a Mongo database.
All collision detection and sprite animation is done with the Phaser.js library

Built Using: Phaser.js, Node.js, and MongoDB
